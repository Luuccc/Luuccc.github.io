I was asked to produce a tester program to send commands via SPI to a TLC59281 (LED Driver), drawing to four seven-segment displays.

This involved building 32-bit payloads from the serial data received that specify the segments to illuminate, the colour bits, and symbol bits. As well as creating a serial transfer method to send payloads to the LED driver chip.

Continuing this, I realized it would instead be useful to refactor my code and build a C++ library. Allowing anyone to interface with this display without needing to understand the payload. I also extended the functionality of the library by adding various helper methods, allowing a user to perform more actions with the display.
